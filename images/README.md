# 图片资源文件夹

这个文件夹用于存放应用程序的图标和图片资源。

## 需要的图标文件

- `add.png` - 添加按钮图标
- `calendar.png` - 日历图标
- `stats.png` - 统计图标
- `edit.png` - 编辑图标
- `delete.png` - 删除图标
- `back.png` - 返回图标
- `chevron_left.png` - 左箭头图标
- `chevron_right.png` - 右箭头图标
- `restaurant.png` - 餐食图标
- `payments.png` - 支付图标
- `trending_up.png` - 趋势图标
- `add_photo_alternate.png` - 添加图片图标

## 图标要求

- 建议使用PNG格式
- 尺寸建议为24x24或32x32像素
- 支持透明背景
- 使用Material Design风格

## 获取图标

可以从以下来源获取图标：
1. Material Design Icons (https://material.io/icons/)
2. Feather Icons (https://feathericons.com/)
3. 其他开源图标库

## 注意事项

- 确保图标版权合法
- 图标风格保持一致
- 文件名要与代码中的引用一致
